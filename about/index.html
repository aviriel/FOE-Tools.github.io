About FOE Tools

General information

Forge of Empires, FOE, and all related trademarks and content belong to InnoGames GmbH.
FOE Tools is an unofficial third party site and is not affiliated with InnoGames.

FOE Tools was created by the French player Yoratheon.

The site is OpenSource, the code is freely available on GitHub.
Thus, everyone with the necessary skills can participate in the improvement of this site.

Cookies

To improve your comfort of use, the site uses cookies.

We store the display language.
In "GB Investment" Tool, we store for each GB the level and various percentages of investors so we do not have to re-enter them each time.
Please note that when you visit a "permalink" page, no information on the level of GB and the various percentages of investors is stored.

Development information

The site is a simple static site hosted on GitHub.

The development is done in Node.js using Marko, a template engine for reactive component developed by eBay. The style is made using Bulma.

An idea of functionality? A problem ? Feel free to create an issue on GitHub or send me an email (in English or in French) at foe-tools@protonmail.com.

Note: Regarding issues on GitHub, please make sure to write them in English to facilitate understanding for the greatest number.
