FOE Tools

A set of tools to simplify life on Forge Of Empire

Information

The site currently has a tool to calculate the number of points a player must put on his GB to secure the first 5 places following a rate (arc bonus).

A second tool is available, "Secure place". This tool calculates the number of FP that a player must put on a GB to block his place. It is integrated in "GB Investment" and also has a standalone version.
